About

The person behind the work

I'm Morney Deetlefs, based in South Africa. I build web platforms, tools and systems for community organisations, events and local businesses — the kind of projects that make a real difference to real people without requiring enterprise budgets.

My approach is practical: use the best free tools available, write clean code that non-developers can maintain, and build things that work reliably at community scale. No overengineering. No unnecessary dependencies. No monthly costs.

Every project I ship is live, used, and solves a genuine problem. The silent auction platform raised funds for a local charity event. The next project will do something equally concrete.

01
Zero-cost first
Community organisations shouldn't pay SaaS fees for tools that could be free. Every project targets R0 running cost.
02
Real deployment
No demos, no concept projects. Everything ships to a live URL, used by real people at real events.
03
Maintainable by anyone
Google Sheets as a database means a non-developer can update content. No server access required.
04
Craft over speed
Details matter. Typography, transitions, error messages — every pixel is considered.

The best tool is the one that costs nothing to run, anyone can maintain, and actually gets used.
